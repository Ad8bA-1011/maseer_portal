# 🌐 Maseer Portal

Client registration portal for Maseer Media's AI video marketing service.

## 🚀 Live URL
https://ad8ba-1011.github.io/maseer_portal/

## ✨ Features
- Trilingual: English, Persian/Dari, Pashto
- Real-time color psychology suggestions
- Secure GitHub Issues integration
- Mobile-responsive RTL design
- Zero external dependencies

## 🏗️ Architecture
- **Frontend:** GitHub Pages (static)
- **Backend:** GitHub Actions (via Issues)
- **Security:** No tokens exposed in client-side code

## 📁 Structure
- `index.html` - Registration form
- `success.html` - Confirmation page
- `assets/js/` - JavaScript modules
- `assets/css/` - Stylesheets (inline for performance)
